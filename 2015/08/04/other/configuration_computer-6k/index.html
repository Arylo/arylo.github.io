F

今晚高中同学老简找我要份电脑配置, 这个… 你不能找我要就有, 你要给点时间, 我要大家知道, 我配的是这个样子, 你们配得也是这个样子(泥垢了

N

要求可不低啊,

  • 6k左右
  • Intel E3 1231 V3
  • N卡 GTX 970
  • SSD要120G, 最好是三星或者闪迪

N

单看6K就觉得剩很多, 但显卡就占了三分之一了…CPU+SSD+硬盘又没有了2k..剩下可选的真的不多
